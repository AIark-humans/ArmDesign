# ArmDesign

Designs for robotic arm

## Overview

ArmDesign is a repository containing CAD designs and documentation for robotic arm systems.

## Contents

- **CADFiles/** - CAD design files for the robotic arm components and assemblies
- **images/** - Visual documentation, reference images, and design illustrations

## Getting Started

1. Browse the `CADFiles/` directory for design files
2. Check the `images/` folder for visual guides and design references
